Node.js

Simple HTTP Server

This is an example of a minimal Node.js server using Express.

// server.js
const express = require('express');
const app = express();
app.get('/', (req,res) => res.send('Hello from Node'));
app.listen(3000);
API

Fetch from Frontend

Use fetch() to call backend endpoints.

fetch('/api/events')
  .then(r => r.json())
  .then(data => console.log(data));

Creating a Route

app.post('/api/contact', (req,res)=>{
  // process req.body
  res.json({ success:true });
});

In-memory vs Database

In-memory: easy for demos. DB (Mongo) required for persistence across restarts.
